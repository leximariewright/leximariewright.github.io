Mira HQ

Tasks

MEASURE WEATHER

The player must press a button that will measure the weather. This process lasts for 5 seconds, and it is located on the Balcony in MIRA HQ.

BUY BEVERAGE

Located in the Cafeteria, the player must order an item from a vending machine using a keypad to select the correct item. The player's exact item needs to order is shown on a piece of paper located under the keypad. The vending machine separates its items by a letter followed by a number (Example: B2). If the player enters the "code" for the wrong item, the vending machine will dispense the item out, allowing the player to attempt to enter the right code again.


WATER PLANTS

This task has two parts. One in the storage, the player needs to locate a watering can and click on it to collect it. After that, the player must head to the Greenhouse to complete the second part of the task. The player needs to water plants at the Greenhouse by clicking on each one of them.

RUN DIAGNOSTICS

Simply interact with the panel and click on the spacebar to begin running diagnostics. After 90 seconds, you must come back to the panel and click on the anomaly, represented by the red circle.


SORT SAMPLES

Sorting Samples is a task located in the Laboratory; it is on the bottom table. This task consists of the player correctly organizing six items into three boxes, separated into three categories. If a player puts an item in the wrong category, the item will gain a red outline indicating that it's wrong. If the player puts the item correctly, the outline will be blue.

ENTER ID CODE

Entering the ID Code is a task located in the admin room next to the map. In this task, players must enter a code on a keypad to complete it. The code is a five-digit number from a card inside a wallet. If the player enters the right code, it will display "OK." On the other hand, if the player enters the wrong code, it will display "BAD."
